🏦 Banking Dashboard - End-to-End Data Analysis Project

An interactive Banking Dashboard built using Power BI, covering the complete data analysis lifecycle — from data extraction to visualization and insights.

📌 Project Overview

This project demonstrates how raw banking data can be transformed into meaningful insights using SQL and Power BI.

Workflow:

Data ➡️ MySQL ➡️ Data Cleaning & Preparation ➡️ EDA ➡️ Power BI Dashboard
📊 Dataset Information
Total Columns: 24
Database: MySQL
Data Type: Banking customer data (accounts, balances, demographics)
🔧 Steps Involved
1️⃣ Data Cleaning & Preparation
Categorized Income into bands:
Low
Mid
High
Standardized categorical variables:
Gender
Nationality
Created conditional columns in Power BI:
Income Bands
Transformed branch codes:
'1', '2' → Branch Names
Mapped Gender:
'1' → Male
'2' → Female
2️⃣ Exploratory Data Analysis (EDA)
🔹 Categorical Analysis:
Gender distribution
Nationality distribution
🔹 Numerical Analysis:
Credit Card Balance
Bank Loans
Bank Deposits
Checking Account
Saving Account
Estimated Income
Superannuation Savings
3️⃣ Key Insights
📈 Strong positive correlation observed between:
Bank Deposits
Checking Account
Saving Account
Foreign Currency Account
💡 Customers with high balance in one account type are likely to maintain high balances across multiple accounts.
